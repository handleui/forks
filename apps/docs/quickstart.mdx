## Installation

Install Detent CLI globally:

<CodeGroup>
```bash curl
curl -fsSL https://detent.sh/install.sh | bash
```

```bash npm
npm install -g detent
```

```bash homebrew
brew install handleui/tap/detent
```
</CodeGroup>

## Basic Usage

### 1. Run a Check

Navigate to your repository and run:

```bash
detent check
```

This parses your CI configuration and runs the relevant workflows against your changed files.

### 2. Heal Failures

If checks fail, let AI fix them:

```bash
detent heal
```

Detent will analyze the failures and apply fixes automatically.

### 3. Verify

After healing, run check again to verify:

```bash
detent check
```

## Example Workflow

```bash
# Make some changes
git checkout -b feature/my-changes
# ... edit files ...

# Run CI locally
detent check

# If failures, heal them
detent heal

# Verify fixes worked
detent check

# Commit and push with confidence
git add . && git commit -m "feat: my changes"
git push
```
